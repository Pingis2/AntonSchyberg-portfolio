Hey there, I'm Anton Schyberg, a 22-year-old student studying Front End Development at Medieinstitutet.


I'm all about infusing creativity into everything I do, whether it's making music, sketching art pieces, or crafting captivating web designs. Long-term projects are my favorite, I love the long process because it keeps me busy, and when I reach the end of a project I start a new one to keep my mind going.


I'm a loyal team player but also enjoy the focus and freedom of solo work. English and Swedish flow effortlessly for me, allowing me to connect and collaborate across different languages and cultures.


While CSS is where I really shine, I'll admit that JavaScript is my current weak spot. But hey, I'm not one to shy away from a challenge. I'm all about learning and growing, and I'm determined to sharpen my JavaScript skills and become the best Front End Developer I can be. Bring on the coding adventures!

Gridpainter

This is a group project that was made during our API class. It was created with API, vanilla JavaScript, SCSS and Socket.io

The point of this project was to create a live game where max 4 people could play at the same time. Our game was a coloring game where you're shown an image and you and your team is supposed to recreate that image with the color that you've been given. There is a 10 minute timer and if you finish before the timer is over, you will be shown the correct image and the image you just did and it will compare them in percent.
